# WizUP University Portal 

Welcome to the **WizUP University Portal** repository! This is a cutting-edge, next-generation web application designed to deliver a unified, seamless, and highly personalized digital campus experience.

WizUP provides an intelligent and aesthetically pleasing platform to connect Students, Faculty, HODs (Head of Departments), and Administrative personnel under a single centralized dashboard experience.

---

##  Key Features

### Role-Based Portals
*   **Student Portal**: Access attendance records, internal marks, leave applications, view timetables, and download study materials securely.
*   **Faculty Portal**: Mark student attendance, grade assignments, track workloads, and independently post course-wide announcements.
*   **HOD Portal**: Comprehensive department analytics, faculty management, leave approvals, and detention monitoring.
*   **Admin Portal**: Complete system configuration, security logs, and role management.

### Modern & Premium Design Language
*   **Glassmorphic Aesthetic**: A slick, semi-transparent user interface layered precisely over dynamic gradient backgrounds.
*   **Micro-Interactions**: Hover, click, and loading states are highly animated to provide an engaging layer of responsiveness.
*   **Contextual Overlays**: Fully structured animated modal systems for information layers like "Help Desks," "About," and "Announcements."

## Technology Stack

1.  **Core Interface**: HTML5, Vanilla JavaScript (ES6+), and pure CSS structural frameworks.
2.  **Styling**: Pre-processed Tailwind CSS for rapid, strict styling combined with custom glassmorphism.
3.  **Visualizations**: **Chart.js** included natively via CDNs to provide detailed dashboard analytics.
4.  **Database & Authentication**: Connects dynamically with **Supabase** via external CDN imports for fast, un-opinionated backend services.

## How to Run Locally

You do not need to install complex dependency paths to run the interface locally:

1.  Clone this repository to your local machine:
    ```bash
    git clone https://github.com/Gaurav1764/WizUP.git
    cd WizUP
    ```
2.  Open the directory in your preferred IDE (like **Visual Studio Code**).
3.  Start a local development server (Using the **Live Server** extension is highly recommended) and point it to the root directory where `index.html` is located.
4.  Navigate to `http://localhost:5500` to start exploring the gateway.

> **Note**: Avoid opening the HTML file directly through the file-explorer (`file:///...`) if you run into Strict-CORS issues. Always prefer hosting it through an isolated web-server locally!
